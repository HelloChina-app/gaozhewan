## 本周精选

- **把 agent 塞进手机键盘**——本周新奇度最高梯队的信号（novelty 8.5）：新加坡团队 Acti 上线 agentic keyboard，长按一个键就能跨 app 跑完多步任务，核心的 Skills 功能用自然语言就能编，早期内测两周建了 1000+ 个。值得写：这是「AI 入口」之争的新样本，写给想把 AI 变成随手动作的普通人。
- **中文字终于能排进 AI 图里**——本周传播潜力最高梯队（viral 8.5）：Google 的 Nano Banana Pro（Gemini 3 Pro Image）是首个能稳定渲染「图内文字」的模型，2–5 秒出 4K、支持 5 个主体一致性。值得写：海报 / 封面 / 电商图第一次有了靠谱的一站式工具，和 Reve 2、Ideogram 4 做横评最有看点。
- **Google 开源了一个「会扩散」的语言模型**——本周最新奇（novelty 9.0）：DiffusionGemma 26B-A4B（Apache 2.0）不再逐字生成，而是并行「去噪」一整块画布，专用 GPU 上最高快约 4 倍，还能本地跑。值得写：扩散式文本模型第一次以 Gemma 之名落地，适合写一篇「自回归 vs 扩散」的科普。
- **给 AI agent 装上看全网的眼睛**——本周综合分最高梯队：国产 CLI Agent-Reach 让任何 agent 读 X / Reddit / YouTube，以及小红书、B站、知乎等 16+ 平台，零 API 费，star 已涨到约 3.7 万。值得写：最适合中文创作者上手的「自动选题调研」工具，实测它怎么把十分钟调研自动化。
- **把 Claude Code 变成一整个视频制作厂**——OpenMontage 冲上 GitHub Trending 第 1，内置 12 条 pipeline、52 个工具、500+ agent skills，用大白话描述就自动完成调研 → 脚本 → 素材 → 剪辑 → 合成，素材链路可全免费本地跑。值得写：把「agentic coding」外延到视频，做一期「我让 agent 剪了条视频」的实测。
- **20 万星的 CLAUDE.md「进化」了**——署名 Karpathy 的十条规则版在 X 流传（真实性未经本人确认），把重点从「怎么写代码」转向「agent 怎么监控自己」，还给出四种该立刻停手的失败模式。值得写：给所有在用 coding agent 的人一份「护栏清单」，逐条翻成普通话。
- **把「接近 Opus」的 agent 能力打到 $2**——Claude Sonnet 5 发布，SWE-bench Pro 拿 63.2、带 1M 上下文，引入价 $2/百万 输入，并成为 claude.ai 免费版与 Pro 版的新默认模型。值得写：给读者一份「便宜跑 agent 到底能干嘛」的实操，和 GPT-5.5 做诚实对比。
- **Agent Skills 从「Claude 的功能」长成行业标准**——anthropics/skills 已约 15.5 万 star，Skill 格式作为开放标准发布后，已有约 40 个客户端接入（Copilot、Cursor、Codex、Gemini CLI 等）。值得写：把「Skill」讲成跨工具的通用能力，手把手教读者写自己的第一个 skill。
- **一台 16G 笔记本就能跑「全模态」**——Gemma 4 12B Unified（Apache 2.0）用无编码器统一架构，是首个原生支持音频输入的中等体量模型，16GB 显存即可本地处理音视频。值得写：门槛极低的「不上云、不烧 API」本地多模态上手教程。

## 这周怎么用

本周的暗线很清楚：agent 正在从聊天框走进你每天在用的入口——键盘（Acti）、视频工作室（OpenMontage）、以及跨工具的通用能力层（Agent Skills）。如果只做一件事，建议这样排主线：

- 主线一（趋势稿）：把 Acti、OpenMontage、Agent Skills 三条拼成一篇「2026 下半年，AI 不再是一个网站，而是嵌进每件工具的一层」的入口之争长文，落点是普通人该怎么重新选工具。
- 主线二（省钱实测）：Sonnet 5 负责跑、Agent-Reach 负责零成本取材、Gemma 4 12B 本地兜底，串一条几乎零成本的 agent 工作流，做成一期「便宜又能打」的教程，传播和实用兼顾。
- 主线三（出图选型）：单独把 Nano Banana Pro 的「图内中文字」拎出来做「封面工厂」教程，是本周最容易转化成实操流量的角度。
